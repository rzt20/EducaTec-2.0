# 📱 EducaTec 2.0 - Frontend

EducaTec 2.0 es una aplicación móvil educativa enfocada en el aprendizaje interactivo. Este repositorio contiene el desarrollo del frontend, construido en Android Studio utilizando Kotlin.

---

##  Tecnologías utilizadas

- Kotlin  
- Android Studio  
- XML (interfaces de usuario)  

---

##  Funcionalidades principales

- Interfaz de usuario intuitiva  
- Navegación entre pantallas  
- Visualización de contenido educativo  
- Interacción con el usuario  

---

##  Integración

Este proyecto corresponde únicamente al frontend de la aplicación.  
El backend se encuentra desarrollado de manera independiente y se encarga de la gestión de datos y lógica del sistema.

---
